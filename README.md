# Rodeo Event Action Plan Generator

A simple MVP web app for creating event action plans for Rodeo Tex Mex.

## What it does

- Create and save events
- Dynamic questionnaire based on event modules
- Supports drink tickets/wristbands, buffet, free flow, canapes, registration and DJ
- Auto-calculates drink tickets and wristbands
- Generates an editable event action plan
- Saves events in browser localStorage
- Exports PDF using browser Print > Save as PDF

## How to run

Open `index.html` in your browser.

No installation needed.
